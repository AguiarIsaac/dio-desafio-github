O Git é um sistema de controle de versão distribuída de código aberto e gratuito projetado para lidar com tudo, 
desde projetos pequenos até grandes com velocidade e eficiência.

Git é fácil de aprender e tem uma pequena pegada com desempenho rápido de raios. 
Ele supera ferramentas SCM como Subversão, CVS, Perforce e ClearCase com recursos como ramificações 
locais baratas,áreas de estadiamento convenientese vários fluxos de trabalho.


Fonte: https://git-scm.com/
